Don't use arrange. Don't put everything in variables.
Use several $: entry points. _$ = mute, s$ = solo
all => all.punchcard()

Randcat to pick random vars.

Maybe start performance with blank screen, get something going... then paste pre-built bits into it.

How to turn a static 'arrange' track into a usable template to play to an audience?
Take the arrange and turn it into a big stack. Then start with it all commented out.

Practice speed typing the whole thing to get a reliable, predictable base that you can fall back on.
Starting with a blank canvas doesn't mean starting with a blank mind. Memorise a pattern that works for you.

Don't forget drums, bassline, chords, melody.
Set up irregular offsets so things roll in and out of hearing.
Gain with euclids.
